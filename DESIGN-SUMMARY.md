# Design summary

- Design concept: Philippines talent atlas
- Layout personality: organic and natural
- Typography: Recoleta headings with Nunito body text
- Color palette: primary #14532d, secondary #86efac, accent #f59e0b, background #f7fee7, text #1c1917, muted #64748b
- Shape language: Large rounded cards
- Visual motif: island map labels and soft blocks
- Unique layout choices: first sections use Talent atlas, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
